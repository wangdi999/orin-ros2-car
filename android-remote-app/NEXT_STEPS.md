- IP 和端口输入
- WebSocket 连接
- 方向控制
- 速度调节
- 松手即停

- 科技风深色仪表盘界面
- 低速、巡航、高速快捷档位
- 链路状态、模式、发送计数、运行日志
- 地图显示、视频流区域、AI 检测框、虚拟摇杆、心跳保护、报警列表
- 使用说明和测试清单

当前阻塞点是：

```text
Android Studio Gradle 同步失败
```

这是环境下载问题，不是 App 功能代码问题。

## 第一步：解决 Gradle 同步

按 `GRADLE_SYNC_FIX.md` 处理：

1. 下载或解压 Gradle。
2. 在 Android Studio 中指定本地 Gradle。
3. 重新执行 Gradle Sync。

完成标志：

```text
Android Studio 不再显示 Gradle project sync failed
```

## 第二步：创建运行配置

同步成功后：

1. 点击右上角 `Add Configuration`。
2. 选择 `Android App`。
3. Module 选择 `mobile`。
4. 保存配置。

完成标志：

```text
右上角出现 mobile 运行配置
```

## 第三步：安装到手机

1. 手机开启开发者模式。
2. 手机开启 USB 调试。
3. 用数据线连接电脑。
4. 手机上点击允许 USB 调试。
5. Android Studio 顶部选择手机设备。
6. 点击绿色运行按钮。

完成标志：

```text
手机上出现并能打开“智能小车遥控”
```

## 第四步：离线功能检查

即使小车暂时不在，也可以先检查：

1. App 是否能正常打开。
2. IP 和端口是否能输入。
3. 速度滑条是否能拖动。
4. 未连接时方向按钮是否不可用。
5. 点击连接失败时是否有提示。
6. 使用 `OFFLINE_TEST_WITH_MOCK.md` 中的模拟服务验证控制消息。

完成标志：

```text
App 在手机端能正常显示和操作
```

## 第五步：连接小车测试

小车到位或通信环境可用后：

1. 手机和小车连接同一个网络。
2. 小车端启动通信服务。
3. App 输入小车 IP。
4. 端口使用 `9090`。
5. 点击连接。
6. 测试方向控制。
7. 测试松手即停。
8. 测试急停。

完成标志：

```text
手机按键能控制小车运动，松开后停止，急停有效
```
