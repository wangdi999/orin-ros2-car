# 智能小车安卓遥控 App

这是成员 C 的安卓手机端遥控应用工程。它用于在安卓手机上连接智能小车，并完成基础遥控、速度调节和急停操作。

本模块只负责 **安卓 App 与小车控制通信**，不包含其他成员负责的功能内容。

## 四、如何打开工程

1. 打开 Android Studio。
2. 点击 `Open`。
3. 选择文件夹：

```text
D:\D\dasanxia\carproject\app
```

4. 等待 Gradle 自动同步。
5. 如果提示安装 Android SDK 或 Gradle 插件，按提示安装。

## 五、如何安装到安卓手机

1. 用 USB 连接安卓手机和电脑。
2. 手机开启开发者模式。
3. 手机开启 USB 调试。
4. Android Studio 顶部选择你的手机设备。
5. 点击运行按钮。
6. App 会安装到手机上，名称为：

```text
智能小车遥控
```

## 六、小车端前置条件

手机 App 通过 WebSocket 连接小车端通信服务。小车端需要提前启动对应服务，并保证能够接收控制指令。

默认端口：

```text
9090
```

App 当前发送的控制话题：

```text
/cmd_vel
```

控制消息类型：

```text
geometry_msgs/msg/Twist
```

## 七、手机连接步骤

1. 手机和小车连接到同一个 Wi-Fi 或热点。
2. 打开 App。
3. 在 IP 输入框中输入小车 IP，例如：

```text
192.168.43.205
```

4. 端口输入：

```text
9090
```

5. 点击 `连接`。
6. 显示 `已连接` 后，方向按钮会变为可用。
7. 按住方向按钮，小车开始运动。
8. 松开方向按钮，小车停止。
9. 需要紧急停止时，点击 `急停`。

## 八、安全建议

- 第一次测试时，建议把小车四轮悬空。
- 速度先设置到 20% 到 30%。
- 先测试 `急停`，再测试方向按钮。
- 不要在人多、空间狭窄或有易碎物品的地方直接落地测试。
- 如果小车方向和按钮不一致，需要调整代码中对应方向的速度正负号。

## 十、当前限制

- 需要小车端通信服务已经启动。
- 需要手机和小车处于同一网络。
- 当前 App 只负责遥控，不展示视频画面。
- 当前 App 不保存历史控制记录。
- 小车实际方向可能受底盘控制程序定义影响，需要实车测试后微调。
- 当前 App 不依赖 OkHttp 等第三方网络库，减少 Gradle 同步时的额外下载。
